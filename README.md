# Exemple de README

Això és un [enllaç](https://google.com) i això són **negretes** i *cursiva* i ara ve una llista:

* hola
* que
* tal

Potser vols una llista numerada:

1. tot
2. bé
3. gràcies

## Un títol de nivell 2

I un troç de codi

    apt-get install virus

```python
a = 2 + 3
```
